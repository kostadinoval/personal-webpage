Current Projects

Logistics management system

This project is for a family friend who wants a system that will allow him to manage his small logistics business in the US. It will focus on having an admin dashboard from which all operations will be managed. Users will be given their login details which will enable them to use the system. Once logged in, they'll be able to see full details of new deliveries, location based tracking and directions, and a full history of completed operations.

This project is active and is in the early stages of development.

University Projects

Django Adsys

A web advertising system created using the Python based Django framework as part of a university assignment. Allows users to create text based adverts, ad spaces and register domains. Using the Scrapy (web crawling) framework the system processes the contents of each page. A separate script calculates a score which indicates the relevance of registered adverts to each of the pages.

This project has not been maintained since its submission.

Python, Django, Scrapy, JavaScript, Bootstrap, HTML, CSS

Github repository

CodeIgniter QA

A questions and answers website created using the PHP based CodeIgniter framework as part of a university assignment. Besides posting questions (which include categories and tags) and answers, as well as basic user profiles, users can also rate other answers based on how useful they are. Users can find questions using a keyword based search or they can browse through questions based on a category.

This project has not been maintained since its submission.

PHP, CodeIgniter, MySQL, JavaScript, AJAX, HTML, CSS

Github repository
